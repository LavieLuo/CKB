## Dataset
- The datasets should be placed in `./dataset`, e.g.,

  `./dataset/OfficeHome`

  `./dataset/ImageCLEF`

- The structure of the datasets should be like
```
OfficeHome (Dataset)
|- Art (Domain)
|  |- Alarm_Clock (Class)
|     |- XXXX.jpg (Sample) 
|     |- ...
|  |- Backpack (Class)
|  |- ...
|- Clipart (Domain)
|- Product (Domain)
|- Real_World (Domain)
```

```
Image-CLEF (Dataset)
|- I (Domain)
|  |- aeroplane (Class)
|     |- XXXX.jpg (Sample) 
|     |- ...
|  |- bike (Class)
|  |- ...
|- P (Domain)
|- C (Domain)
```
